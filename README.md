# News List APP

## Description

Using this project you can Adding, Editing , deleting and Searching of News witch you created

## Installation

```bash
$ git clone https://github.com/AAmiryan/news-app.git
```

navigate to root directory

```bash
$ cd news-app

```

Install project

```bash
$ npm install
$ yarn
```

## Running the app

```bash
$ npm run dev
$ yarn dev
```

## Running the server

```bash
$ npm run start:json-server
```

## Technology

- React
- Typescript
- Redux Toolkit
- Redux-Saga
- Antd
- React-Router-Dom
- Json-Server
